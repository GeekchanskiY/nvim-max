# Shortcut list which makes vi goat 

### Editor 
#### default mode 
`*` - search for word under cursor
#### insert mode 
`alt+A` - going to the end of line wihout exiting insert mode 
`alt+o`, `alt+O` - newline 
`alt+p` - paste 



### useless, but fun 
`ctrl+a`, `ctrl+x` on a number - increment or decrement it.
